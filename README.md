# CharkLi Site

A basic Node.js website using TypeScript and SCSS.

## Setup

1. Install dependencies: `npm install`
2. Build the project: `npm run build`
3. Start the server: `npm start`

For development: `npm run dev` to run with watch mode.

## Technologies

- Node.js
- TypeScript
- Express
- SCSS (compiled with Sass)# CharkLi
